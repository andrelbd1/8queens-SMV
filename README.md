# 8queens_SMV
Modeling the 8queens problem in SMV

The eight queens puzzle is a problem of placing eight chess queens on an 8Ã—8 chessboard so that no two queens threaten each other. Thus, a solution requires that no two queens share the same row, column, or diagonal.


To run ./NuSMV <file>  
ex: 
  ````bash
  ./NuSMV 8queens.smv
  ````
